## Description
Donne la météo du jour.
Nécessite une clé wunderground (gratuite).
Obtenir une clé: https://www.wunderground.com/weather/api/

## Usage
```
Vous: Quelle est la météo?
Jarvis: je regarde...
Jarvis: Ciel plutôt dégagé. Minimales : 9
```
